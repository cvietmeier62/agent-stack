# Research Analyst Agent

AI research analyst with deep research, source evaluation, and synthesis reporting.

## Skills

- `/deep-research [topic]` — Comprehensive research with panel quality gate
- `/source-evaluation [url]` — Score a source's credibility
- `/synthesis-report` — Combine findings into structured report

## Quick Start

```bash
cd ~/agent-stack/examples/research-analyst
claude
# Ask: "Research the current state of AI agent frameworks"
```
