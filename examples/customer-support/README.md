# Customer Support Agent

Customer support agent with ticket handling, escalation, and daily reporting.

## Skills

| Skill | Command | Description |
|-------|---------|-------------|
| Handle Ticket | `/handle-ticket` | Process a ticket end-to-end: KB search, draft response, quality panel, send or escalate |
| Escalation Check | `/escalation-check` | Evaluate a ticket against 5 escalation triggers with scoring |
| Daily Summary | `/daily-summary` | End-of-day metrics report with trends and recommendations |

## Setup

1. Open this folder in Claude Code
2. Add your knowledge base articles to `data/kb.json`
3. Add ticket files to `data/tickets/`
4. Adjust SLA targets and routing in `data/strategy.json`

## Data Files

- `data/strategy.json` — SLA targets, escalation thresholds, tone preferences, auto-close rules
- `data/kb.json` — Knowledge base articles (create this with your content)
- `data/tickets/` — Ticket JSON files (one per ticket)
- `data/templates.json` — Response templates by category (create this with your templates)
- `data/summaries/` — Daily summary reports (auto-generated by `/daily-summary`)
